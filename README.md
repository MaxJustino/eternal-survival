## 📌 About the Project | Sobre o Projeto

🇺🇸  
**Eternal Survival** is a top-down survival game being developed using GameMaker.  
The goal is to design scalable systems while applying clean coding practices and proper version control workflow.

🇧🇷  
**Eternal Survival** é um jogo survival top-down desenvolvido no GameMaker.  
O objetivo é criar sistemas escaláveis aplicando boas práticas de código e fluxo profissional de versionamento.

---

## 🚀 Current Implementations | Implementações Atuais

- 8-direction player movement (normalized diagonal speed)
- Idle and walk animation system
- Viewport scaling (320x180 base resolution → 1280x720 output)
- Basic project architecture organization
- Git version control integration
