# 🎨 Rendering Setup | Configuração de Renderização

🇺🇸  
The project uses a **pixel-art friendly rendering setup** to maintain visual clarity.

The game runs with a **base resolution of 320×180**, which is scaled to **1280×720 (4x scale)** during gameplay.

To preserve the sharpness of the sprites, the **"Interpolate Colors Between Pixels"** option was disabled.

This ensures that pixels scale proportionally without smoothing, maintaining a **clean and crisp pixel-art look**.

🇧🇷  
O projeto utiliza uma **configuração de renderização adequada para pixel art** para manter a nitidez visual.

O jogo roda em uma **resolução base de 320×180**, que é escalada para **1280×720 (escala 4x)** durante a execução.

Para preservar a nitidez dos sprites, a opção **"Interpolate Colors Between Pixels"** foi desativada.

Isso garante que os pixels sejam ampliados proporcionalmente sem suavização, mantendo um visual **nítido e fiel ao estilo pixel art**.
